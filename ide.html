Présentation des IDE

ATOM

Présenté comme l'éditeur de texte du XXIème siècle par ses développeurs, Atom est un puissant éditeur de code open source dont la structure repose sur les meilleurs éléments d'autres éditeurs de texte également open source.

SUBLIME TEXT

Sublime Text est un éditeur de texte source et de code source fonctionnel écrit en C ++ et Python. Sublime Text propose aux développeurs un éditeur de texte qui se démarque des autres par son interface et ses fonctionnalités.

VISUAL STUDIO

Visual Studio est un ensemble complet d'outils de développement permettant de générer des applications web ASP.NET, des services web XML, des applications bureautiques et des applications mobiles.
